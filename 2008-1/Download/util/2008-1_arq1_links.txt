Links recomendados

http://sourceforge.net/projects/ivi/
    Icarus Verilog Interactive compiler & simulator     
    Versão recomendada: 0.4-pre-20031121 [Notes] (2003-11-22 00:00) 
    Pacote para Verilog e para visualização de ondas
    Obs.: As versões mais novas dependem do Eclipse,
          por isso recomenda-se uma versão mais antiga.
          O ambiente para edição e compilação pode ser o JGrasp.
          Para tanto, seguir o roteiro de configuração em
          JGrasp_Verilog.txt e os arquivos com templates.
          Copiar os arquivo ivi.bat run_ivi.bat 
          para a pasta IVI\bin
    Obs.: O programa também pode ser encontrado para cópia local em 
          ftp://ftp.pucmg.br/pub/Compilador/Programacao_Hardware/ISL/IVI_0.4-pre-20031121.exe

http://sourceforge.net/projects/circuit
    Editor e simulador de circuitos lógicos
    Versão recomendada: 2.1.6 [Notes] (2007-05-07)  
    Obs.: O programa também pode ser encontrado para cópia local em
          ftp://ftp.pucmg.br/pub/Compilador/Programacao_Hardware/ISL/LogiSim_v216.exe

http://www.cs.colby.edu/djskrien/CPUSim/CPUSim3.5.1.zip
    Editor e simulador de CPU's em Java

http://www46.homepage.villanova.edu/timothy.m.white/
    Editor e simulador de máquinas de estados finitos
    Versão recomendada: 1.3 [Notes] (2006-05-26) 
    Obs.: O programa também pode ser encontrado para cópia local em
          ftp://ftp.pucmg.br/pub/Compilador/Programacao_Hardware/ISL/JFast_v13.exe

http://www.cs.york.ac.uk/netpro/bboard/
    Simulador de protoboard em Java
    Obs.: Recomenda-se usar a versão em applet.
          A versão disponível para download no site contém erro.
          Uma versão corrigida também pode ser encontrada para cópia local em
          ftp://ftp.pucmg.br/pub/Compilador/Programacao_Hardware/ISL/JBreadBoard.zip


Alternativas:

http://www.icarus.com/eda/verilog/Release_Notes_for_Icarus_Verilog_0_8.html
    Icarus Verilog compiler (sem simulador)     
    Pacote para Verilog
    Obs.: Copiar o arquivo run.bat em 
          ftp://ftp.pucmg.br/pub/Compilador/Programacao_Hardware/Compiladores/
          e colocar na pasta IVerilog\bin

http://context.cx/component/option,com_docman/Itemid,48/   
    Text editor & Verilog highligther (alternativa para o JGrasp)      
    Editor de textos com destaque para palavras chaves em Verilog
    Obs.: O Verilog highligther (VerilogHDL.chl) deverá ser copiado
          em separado e instalado na pasta Context\Highligthers
          Esse arquivo também pode ser encontrado para cópia local em
          ftp://ftp.pucmg.br/pub/Compilador/Programacao_Hardware/Compiladores/Verilog/VerilogHDL.chl

http://www.sutherland-hdl.com/on-line_ref_guide/vlog_ref_top.html
    Verilog reference card
    Manual online para Verilog
    (melhor copiar e instalar na pasta IVI\docs ou IVerilog\docs)
    Obs.: O programa também pode ser encontrado para cópia local em
          ftp://ftp.pucmg.br/pub/Compilador/Programacao_Hardware/Compiladores/Verilog/manual.zip
