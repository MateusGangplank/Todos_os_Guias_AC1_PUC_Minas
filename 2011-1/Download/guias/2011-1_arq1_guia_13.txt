Pontifícia Universidade Católica de Minas Gerais
ARQ1 – Guia 13
Data de entrega: 16-20/05/2011

Tema: Arquitetura de Computadores

Desenvolver programas em Verilog
e relatório de testes.

Proposição básica

01.) Implementar um programa para 
     o processador "ultra" para
     calcular o resultado da expressão aritmética

     03 + 04 * 05

02.) Implementar um programa para 
     o processador "ultra" para
     testar se o resultado da operações entre 
     duas posições na memória de dados
     é igual a zero:
     DICA: Usar o valor igual a zero para falso, 
           e valor igual a um para verdadeiro.
           Usar JUMPZ.

     x1 - x2 == 0

03.) Implementar um programa para 
     o processador "ultra" para
     testar se o resultado da comparação de 
     duas posições na memória de dados
     é maior que zero:
     DICA: Usar o valor igual a zero para falso, 
           e valor igual a um para verdadeiro.
           Usar JUMPS.

     x1 > x2

Extras

04.) Implementar um programa para 
     o processador "ultra" para
     testar se o resultado da comparação de 
     duas posições na memória de dados
     é maior ou igual a zero:
     DICA: Usar o valor igual a zero para falso, 
           e valor igual a um para verdadeiro.
           Usar JUMPS e JUMPZ.

     x1 >= x2

05.) Implementar um programa para 
     o processador "ultra" para
     testar se o resultado da comparação de 
     duas posições na memória de dados
     é menor que zero:
     DICA: Usar o valor igual a zero para falso, 
           e valor igual a um para verdadeiro.
           Usar JUMPS.

     x1 < x2
