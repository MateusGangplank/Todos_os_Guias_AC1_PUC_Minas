Tema: Arquitetura de Computadores

Desenvolver programas em Verilog

A ferramenta para edição, compilação
e testes está disponível em

http://8085simulator.codeplex.com/releases/view/33943


Proposição básica

01.) Implementar um programa para 
     o processador 8085 para
     calcular o resultado 
     da expressão aritmética abaixo,
     usando dados com 8 bits.

     dado04 = dado01 + dado02 - dado03

02.) Implementar um programa para 
     o processador 8085 para
     calcular o resultado 
     da expressão aritmética abaixo,
     usando dados com 16 bits.

     dado04 = dado01 + dado02 - dado03

03.) Implementar um programa para 
     o processador 8085 para
     calcular o resultado 
     da expressão aritmética abaixo,
     usando dados com 8 bits 
     codificados em BCD.


Extras

04.) Implementar um programa para 
     o processador 8085 para
     calcular o resultado 
     da expressão aritmética abaixo,
     usando dados com 8 bits.

     dado03 = 2 * dado01 + dado02


05.) Implementar um programa para 
     o processador 8085 para
     calcular o resultado 
     da expressão aritmética abaixo,
     usando dados com 8 bits.

     dado03 = 2 * (dado01 + dado02)
