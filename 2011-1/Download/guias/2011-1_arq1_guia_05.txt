ARQ1 Guia-05
Data de entrega: 28/02-04/03/2010

Tema: Circuitos lógicos

Os comandos básicos em Verilog incluem:
- definir estrutura de um módulo
- definir métodos principals (init, main_process) 
  com dados e ações a serem executadas
- definir armazenadores de dados (posições de memória): 
  constantes e variáveis
- definir estruturas para auxílio à depuração:
  saídas de registradores e de conteúdos de memória
- definir componentes/módulos

Proposição básica

Guia05 - Capítulo 03
Fazer um programa em Verilog e
simular a expressão resultante em LogiSim,
para atender a cada uma das proposições abaixo.

01.) FAZER um circuito lógico para realizar 
     uma soma completa de dois valores binários 
     com quatro bits cada, usando subcircuitos 
     (FULL_ADDER e HALF-ADDER para o primeiro par).
     Simplificar a expressão lógica equivalente.

02.) FAZER um circuito lógico para realizar 
     uma diferença completa de dois valores binários 
     com três bits cada, usando subcircuitos 
     (FULL_SUBTRACTOR e HALF-SUBTRACTOR 
     para o primeiro par).
     Simplificar a expressão lógica equivalente.

03.) FAZER um circuito lógico para realizar 
     um produto completo de dois valores binários
     com três bits cada.
     Simplificar a expressão lógica equivalente.
     DICA: Usar o circuito do exercício 01.

Extras

04.) FAZER um circuito lógico para realizar 
     uma soma-completa de valores binários 
     com três bits cada,
     usando apenas subcircuitos com portas NAND.

05.) FAZER um circuito lógico para realizar 
     uma diferença-completa de valores binários
     com três bits cada,
     usando apenas subcircuitos com portas NOR.
